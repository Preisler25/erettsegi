Milyen volt a katonai helyzet 1849 elejen
    -> Decemberben Windischgratz megindult magyarorszagra 
    -> Magyar hadsereg nem volt felkeszulve egy negy ossze csapasra az Csaszari sereggel
    -> Gorgei Artur feladta a dunantult 
    -> OHB es az Orszaggyules Debrecenbe koltozott
    -> PestBuda elveszett
    -> batthyanyt januar 5. letartoztattak
    -> Gorgei visszavonult eszakra, ezzel megosztva az keletre nyomulo csaszari sereget
        -> Felvideki hadjarat 1849 januar - februar
            -> Branyiszkoi attores februar 5. (Szembol is csaszari sereggek voltak) 
    -> Gorgei Megerkezett a Tisza menti koncentracios teruletre
    -> Kossuth rivalizalt Gorgeivel
        -> Levaltotta a Sereg vezeteserol
    -> kapolnai casta (dontetlen)
    -> Windischgratz elbizza magat

    Olmutzi oktrojal alkotmany (marcius 4.)
    -> Alkotmanyozo nemzet gyules helyett, kiraly es tanacs adoi
    -> Kenyszeritett mert rajuk lett tolva
    -> Centralizalt birodalmat hozott letre
    -> Magyarok tobbsege ezt nem tudta elfogani

    Erdelyi Hadszinter
    -> Erdelyben allomasozo csaszari csapatok szembe fordultak a magyar vezetessel
        -> csatlakozznak hozzajuk a romanok
            -> az udvar nekik a birodalmon beluli onnalosagot igert
    -> A csaszari es roman seregek egesz erdely megszaltak
    -> csak a Szekely felkeles tuddot ellen allni HaromSzekben
    -> Garbor Aron 
        -> Agyukat ontott
        -> Csata kedvet addot
    -> Bem Jozsef 
        -> februar 9. piski csataban gyozelem
        -> marciusban Pacifikalta erdelyt
