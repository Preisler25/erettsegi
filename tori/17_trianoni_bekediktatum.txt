Létrejötte
    Magyarorszag etnikailag tagolt
    I. Vilaghaborus vereseg
        -> 1918 a nemzetisegek csatlakoztak az anyaorszagukhoz
        -> nemzeti onrendelkezes
        -> modszere a nepszavazas lett volna
        -> a gyoztes nagyhatalmak dontottek

    Francia kulpolitikai koncepcio
        -> francia barat szovetsegi rendszer torve 
            -> Lengyel, csehszlovak, roman, Szerb-Horvat-szloven kiralysag
            -> Ahoz hogy ezek az alamok huek, es erosek legyenek 
                -> megfelelo jutalmat kellet nekik adni
                -> magyarorszag rovassara, hiszen a vesztes fel ugysem tud nagyon ellen kezni 
                -> haborus felelosseget a vesztesekre rottak
            -> Kisantant megalapul (magyar ellenes)

    Versai Beke konferencia 
        -> 1919 tavasz: kozep europai hatarjavaslatok
        -> beke konferencia (csak a gyoztesek vettek reszt, es ok tehettek csak javaslatokat)
        -> Antant a Huszar kormanyt ismerte el, emiatt oket hivtak meg hogy irjak ala a beke diktatumot
            -> 1920 januaraban utaztak ki
            -> Apponyi Albert 
            -> bekedelegacio
                -> nagyon kommoly felkeszules
                    -> "Voros terkep" - Teleki pal
            -> lehetoseget kaptak hogy elmondjak vedo beszedeiket
                -> Apponyi nagy hatasu beszedede 
                -> nem fogadtak el a beke felteteleket
        -> 1920 junius 4. 
            -> Nagy trianoni palota:
                -> Drasche Lazar Alfred es Benard Agoston
                    -> Delegaltak
                    -> Ok ala irtak veggul a beke diktatumok

Hatása

    Nepessegi, Ethnikai 
    -> a nepesseg 60%-a veszett el
    -> 3.3 magyar embert kerult a hataron tulra
        -> minden 3. magyar az anya orszagtol elszakadt
        -> hatarhoz kozeli tombokben eltek
    -> nem tartottak be a kisebbsegvedelmi jogokat
        -> allami intezmenyekben szemelycsere, husegesku
        -> betelepitesi politika
            -> fel lazitani a magyar tomboket
            -> az etnikai viszonyok atrendezese
        -> kozigazgatas atalakitasa
            -> kissebsegi jogok megkerulese
        -> assziminacio az oktatasban
            -> magyar iskolak nem kaphattak allami tamogatast
        -> magyar kulturat csak onerobol lehetett fenttartani
    -> magyarsag aranya csokkenesnek indult a hataron tul
    -> tobb mint 400 000 magyar menekult at Magyarorszagra

    Gazdasagi infrastukturalis
        -> Budapest-tulsuly
        -> videken kozpont hiany
        -> Ha megmaradt vonzaskorzetuk csokkent
        -> Magyarorszagi vasuthalozat jelentosen torzult
        -> Nyersanyag lelohelyek elvesztese
            -> fa 
            -> vaserc
            -> sobanyak
        -> gazdasagi kapcsolatok megszunese
        -> Egyseges vamterulet felbomlasa
        -> kis nemzetisegi, protekcionista piacok
        -> tokebefektetesek csokkentek

    Revizio
        -> Beke dikatatum ujra gondolasa
        -> Nemzeti minimum volt, magyarok szemeben teljesen elfogadhatatlan volt fentallo helyzet
        -> kulpolitikailag komoly kenyszer palyat szabott
        -> Tobbseg a bekes reviziziot tamogatta
        -> legtobben megelegedtek volna az ethnikai (csak magyar teruletek visszacsatolasaval)
        -> de a szelsoseg az integralis az egesz tortenelmi magyar orszag visszaszezeset szerette volna
    
    Revizionizmus
        -> felulviszgalat (bekes vagy fegyveres uton)

    Irredentizmus
        -> egy terulet vissza szerzeset kovetelo mozgalom (etnikai/tortenelmi jogon)

    Revansizmus
        -> viszavagas (fegyveres uton)

    Sopron eset 
        -> Rongyos garda
        -> osztrakok targyalas
        -> vegul nepszavazas
            -> magyarok melleti dontese
