Tatarjaras
    Elozmenyek
        
        -> 4 Bela (1235-1270)
            -> 2 Andras fia 
            -> kiralyi birtokok visszaszerzese
            -> kiralyi tanacs szekeit elegettette, ne lehessen a jelenleteben leulni
            -> kunok betelepitese
                -> barokkal szembeni katonai eronek
                -> nomad, poganyok
            -> komoly feszultseg
        
        -> Julianusz barat
            -> keletre indult 
            -> magyarokat keresni
            -> elso utja sikeres, meg is talalja oket Magna Hungaria teruleten
            -> masodik utja, mongol pusztitast talal mar csak
        
        -> Mongol felemelkedes
            -> kijev eleste (1240)
            -> Batu kan, vezetesevel elerik magyar orszagot

        -> Magyarorszag
            -> Elokeszuletek
                -> 4 Bela osszehivta az orszag haderot
                    -> Nem minden nemes vonult harcba
                        -> 4 bela korabbi intezkedesei miatt
                -> Karpati eroditesek
        
    Tatarjaras (1241-1242)
        -> Muhi donto csata (aprilis 11-12)
            -> szeker varak
            -> mongol letszam foleny
            -> Sajon atkelo mongolokat vissza verik
            -> Masnap hajnalban 
                Eszak felol kerulve sekelyebb ponton atkelve
                bekeritettek a magyarokat
                szeker var mivel nem voltak felkeszulve a magyarokat nehezitette
                4 Bela ephogy megmenekult
        
        -> 4 bela menekult
            -> Dalmaciaig 
            -> itt fogada meg ha magyar orszag tulele lanyat isten szolgalataba adja
        
        -> Mongol kivonulas
            -> nagy kan valasztas?
            -> szakaszos hoditas?
