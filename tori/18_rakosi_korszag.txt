Gazdasag
    Szovijet minta masolasa
        -> Magyarorszag alapbol egy agrarorszag 
        -> ebbol vas es acel orszagat akartak megteremteni
        -> stalini tervek megvalositasa
        -> felkeszules a harmadik vilaghaborura
            -> nehez ipar fejlesztese
        
    A Gazdasagi keretek megvaltoztatasa
        -> tervgazdasag a piac gazdasag helyett
        -> allami tervek keszulnek
            -> mennyit kell kinek hol termelnie
            -> 3-5 eves tervek
        -> allamositas
            1946 -> ipari nagyuzemek
            1948 -> 100fo folotti foglalkoztatott
            1949 -> 10fo folotti foglalkoztatott

    Mezogazdasagi atalakitasok
        -> magan tulajdon megszuntetese
        -> munka ero felszabaditas es at csoportositas
        -> kollektivizalas
            -> termeloszovetkezetek
            -> jobban lehet gepesiteni
            -> tarsadalmi ellenallas
            -> eroszakos eszkozok az ellenallassal
        -> beszolgaltatas
            -> uj ado, irracionalisan magasak
            -> ezzel is kenyszeritettek az embereket a tsz-ekbe
            -> elkeztek az emberek elrejteni a termeseiket
            -> padlas sopres 
                -> avhsok megjelnetek 
                -> es felturtak a hazat
            -> fekete vagas 
                -> nem bejelentett diszno vagas
        -> kulakok (jomodu parasztok) elleni kampany
            -> ellenseg kep kiepitese
        -> mezogazdasagi vissza eses
            -> egyeni erdek megszunese
            -> csak a munka ido szamit nem a minosegi munka
            -> folyamatos elelmiszer hiany alakult ki magyar orszagban
                -> jegyrendszer bevezetese (beke idoben)
        
    Eroltetet iparositas
        -> Nehez ipar fejlesztese 
        -> hianygazdasag
            -> mert a konyu ipar nem fejlodik
        -> nyersanyagszegenyseg
        -> munkaerot a kollektivizalassal akartak biztositani
        -> hatalmas ipari nagyberuhazasok
            -> stalinvaros
            -> Rakosi matyas vas es fem muvek
        -> egyeni erdekekeltseg hianya 
            -> munka versenyek
        -> hiany gazdasag
            -> 1953 gazdasag az osszeomlas szelere jutott
