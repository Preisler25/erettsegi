Mohácsi csata
    
    Elozmenyek
    -> Oszman birodalom terjeszkedes
    -> magyarok felmondtak a torokok beke szerzodes 
    -> I nagy szulejman lepet oszman tronra
    -> Szettagolt Nyugat europai
    -> Gyenge kiralyi hatalom
        -> csokennek a bevetelek
        -> megerosodnek a barok
    -> II. Lajos
        -> 1521 aug 29ben nandorfehervar elsett
        -> Tomori Palt kinevezte kalocsai erseknek
            -> Deli reszek fokapitanya
            -> torok elleni vedelem meszervezese
        -> 1526 Petervarad eleste
        -> Szulejman atkelt a deli vegvarvonalon
    
    Mohacs
        Magyarok 20-25 ezren
        -> Szapojai Janos nem erkeztek meg
        -> Horvat seregek sem erkeztek meg
        -> Nemetromai csapatok sem erkeztek meg

        Oszman hadero
        -> 60 ezer
            -> Szpahik
            -> Janicsarok
        -> legmodernebb tuzerseg

        Csata 1526 aug 29
        -> Egesz nap, faraszto varakozas
        -> foparancsnok Tomori Pal
        -> Kezdetben
            -> Magyar lenduletes tamadas
            -> Janicsaroknal megtort a magyar lendulet
        -> Kesobb torok tulero folenye 
        -> Teljes tragikus vegzet 
        -> 2. Lajos is elhunyt menekules kozben 
        -> Szulejman Budaig vonul csak, majd kivonul az orszagbol
            -> Szakaszos hoditas?

Kettős királyválasztás
    Koznemesseg 
        -> Szapolyai Janos
        -> Magyar kiraly kell
        -> Eros Erintettlen hadero
        -> Rakos mezejen kiralya valasztottak

    Dunantuli Forendenk
        -> Habsburgok Ferdinand
        -> becsujhelyi szerzodes
        -> Habsburg - Jagello hazassagi szerzodes
        -> Pozsonban elso Ferdinado megvalasztottak kiralynak
        -> Nyugat europai segitseg ha jon a torok

    Tronharc
        -> Ferdinand Legyozte Szapolyait 
        -> Szapolyai lengyel orszagba menekult
        -> Ferdinand egyesiti az orszagot

Török hódító hadjáratok 1566-ig
    1529
    -> Szulejmannak nem tetszet hogy ferdinad egyesitette az orszagot
    -> Uj bunteto hadjarat
    -> Szapolyai a deli hataron varat a torokot
    -> Szapolyai vazallussagot fogadott Szulejmannak
    -> Becsig vonult Szulejman, de nem sikerult bevennie

    1531
    -> Szulejman celpontja Becs
    -> 5.Karoly birodalmi haderot gyujtott a vedelemre
    -> Szulejman ezert nem tamadott
    -> Koszeg ostroma
        -> Megalapodas
        -> Torokok kituztek a zaszlot
        -> A varat meghagyta a magyarok kezen 
    
    Egyensuly helyzet
        -> 2 birodalom egyseges erovel bir egyik sem tudja legyozni a masikat

    1538
    -> Varadi egyezmeny
    -> Titkos
    -> Szapolyai es Ferdinand kozott
    -> elismertek egymast kiralynak 
    -> Szapolyai megigerte halala utan az o terulete a Habsburgoke
        -> ekkor meg nem volt utodja
    
    3 Reszre szakadas
    -> varadi egyezmeny felborult
    -> Szapolyainak gyereke szuletett
        -> Janos Zsigmond
        -> Megeskedte legfobb hiveit hogy fianak garantaljak a tront
    -> 1540ben meghalt Szapolyai Janos
        -> Szulejman elinult megszilarditani a magyar helyzetett
            -> Ezert bevonul az orszag kozepso reszere es kozigazgatasa ala vonja
            -> Latogatast tesz a csecsemo Janos Zsigmondhoz
                -> 1541 aug 29
                -> Kozben a torokok baratian bemennek buda varaba 
                -> majd csellel elfoglaljak buda varat 
    -> Eszak Nyugat
        -> Magyar Kiralysag
            -> Habsburg fenthatosag
    -> Kozepso Resz 
        -> Torok Hodoltsag
    -> Keleti orszagresz
        -> Keleti-Magyar Kiralysag/ Erdelyi fejedelemseg
            -> Janos Zsigmond, Frater Gyorgy
    
    Varhaboruk folytatodtak
    -> Frater Gyorgy Keleti kormanyzo
        -> Habsburgok kezere ki akarta jatszani a keleti orszag reszt
        -> Izabellat es Janos Zsigmondot lemondatta
        -> Erdelyt atadta ferdinandnak (1551)
    -> Ferdinandnak nem volt eleg ereje hogy megtartsa az egesz orszagot

    1551 
    -> Frater Gyorgy miutan a torokokkel is megprobalt targyalni
        -> politikai gyilkossag aldozata lett

    1552
    -> Torokok bunteto hadjaratott inditottak
    -> Temesvar eleste
        -> Losonczy Istvan
    -> Eger
        -> Dobo Istvan
        -> elovarak szerepe
            -> feltartottak eleg idejig a torokoket,
            -> igy eger ostromara nem maradt eleg ido

    1556
    -> Janos Zsigmondot vissza hivtak az erdelyiek
    -> Erdelyi Fejedelemseg kialakulasa

    1566
    -> Szulejman utolso hadjarata 
    -> Szigetvar ostroma
        -> Ostrom kozben elhunyt Szulejman
        -> Zrinyi miklos vitezul vedte a varat
        -> utolso erejukkel meg kirohantak a varbol
